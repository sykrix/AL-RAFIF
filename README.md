# فاتورة استراحة الرفيف

موقع ثابت لإنشاء فاتورة حجز بصيغة PDF من المتصفح مباشرة.

## الملفات المطلوبة

- `index.html`: الصفحة الرئيسية والنموذج وتوليد PDF.
- `al-rafif-logo.png`: شعار استراحة الرفيف.
- `logo-data.js`: نسخة مدمجة من الشعار لاستخدامها داخل PDF.
- `.nojekyll`: لضمان عمل الموقع على GitHub Pages بدون معالجة إضافية.

## الرفع على GitHub Pages

1. ارفع الملفات إلى مستودع GitHub.
2. ادخل على Settings ثم Pages.
3. اختر الفرع `main` والمجلد `/root`.
4. احفظ الإعدادات.
5. افتح رابط GitHub Pages بعد اكتمال النشر.

## ملاحظات

- رقم الفاتورة يتولد بتسلسل محفوظ في المتصفح.
- تاريخ الحجز يبدأ بتاريخ اليوم ولا يقبل تاريخاً سابقاً.
- رقم العميل يجب أن يكون 8 أرقام ويبدأ بـ 9 أو 7.
